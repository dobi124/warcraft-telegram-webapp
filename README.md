# Camp WebApp (Battle-camp-like demo)

Оригинальные ассеты и мини-сцена под Telegram WebApp / браузер.  
Запуск: любой статиκ-сервер (например, `python -m http.server`), открыть `/index.html`.

Структура:
- `assets/` – PNG-ассеты (тайлы, спрайты, UI)
- `index.html` – Phaser 3 сцена с композицией, как на скриншоте
